## 如何启动？

``` bash
# 安装依赖
$ npm install

# 启动，并且拥有热重载。（3000端口）
$ npm run dev

# 生产模式编译+运行
$ npm run build
$ npm start

# 生成静态工程
$ npm run generate
```
